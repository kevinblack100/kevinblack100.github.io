  • JSON Web Token Demo
    • Look at the following files for an idea of how JSON Web Tokens could work. There are many other possible ways do it. To experiment with the code you will have to download all of the files into the same directory and then run "node server.js".

    • server.js - Node.js server file
    • index.html - HTML page; note that it is not functional on this site; save the file and access it from your localhost/Cloud9 server
    • client.js - javascript to run in the browser
