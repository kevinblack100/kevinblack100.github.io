Demos / Tutorials

  • JSON Web Token Demo
    • Look at the following files for an idea of how JSON Web Tokens could work. There are many other possible ways do it.
    • Save all of the files into the same directory.
      • package.json - Node.js project package info
      • server.js - Node.js server file
      • index.html - HTML page; note that it is not functional on this site; save the file and access it from your localhost/Cloud9 server
      • client.js - javascript to run in the browser
    • Install node modules:
      • npm install express --save
      • npm install jsonwebtoken --save
    • Run node
      • node server.js
